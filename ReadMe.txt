Prerequisites
#############

1. Python for Windows (tested with Python 3.12 [Windows Store version])
2. Flask (for debugging your web application)

If Flask is not installed:

  Install with the following:
    (from CMD) pip install flask

Debugging environment setup
###########################

1. Create/copy the following files to your working folder, and create your repo

Folder Structure (from repo root)
  [Templates]
  \ index.html
    result.html
  app.py
  ReadMe.txt (this file)

2. From the CMD prompt - cd to your repo root folder and enter:
  python app.py

3. You can now open the Debug web site from a Browswer: http://127.0.0.1:5000/index.html, or //localhost:5000
